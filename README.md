## âœ¨ Features

- **Infinite Whiteboard Canvas**: Pan and zoom across an expansive workspace that scales with your project.
- **Hierarchical Node Management**: Create infinitely nested task relationships or tear them down to keep them as independent root blocks.
- **Smart Auto-Layout**: Powered by Dagre for clean, automated left-to-right organization of complex treesâ€”no more manual realigning.
- **Premium Glassmorphism UI**: Beautifully crafted interface with dark mode support, neon accents, and smooth micro-animations.
- **Interactive Checklists**: Add and manage sub-items effortlessly within any task node.
- **Seamless Connections**: Intuitive drag-and-drop to connect, reparent, or unlink tasks.
- **Quick Duplication**: One-click deep cloning of entire task branches, complete with children and checklists.

## ğŸš€ Getting Started

### Prerequisites

- Node.js 18.x or later
- npm, pnpm, or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/SR-123456789/tree-todo.git
   cd tree-todo/frontend
   ```

2. Install dependencies:

   ```bash
   npm install
   # or
   pnpm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   # or
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
